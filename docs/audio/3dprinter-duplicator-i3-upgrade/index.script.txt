Title: Duplicator i3 3D Printer Upgrade.

TLDR: A how-to on upgrading a Duplicator i3 with modern 3D printer parts, firmware, and creativity. This post is still in progress.

The Duplicator i3 is a common 3D printer that many now consider outdated. Instead of throwing it away and buying a new one, I took the challenge of upgrading and replacing parts to modernize it. There is also a lot of resources and parts still available.

The To-Do List

First, install a glass bed.

Second, change the steppers from 1 over 16 to 1 over 32 steps. This provides cleaner and smoother prints.

Third, replace the existing Melzi board with an MKS Gen 1.4 board. This could have been the standard RepRap Arduino Mega 2560 board as well. Regarding this replacement, there are several sub-steps. First, use JST 2-pin to 3-pin adapters for endstop connectors. Second, add a JST connector for the extruder stepper. Third, make Marlin Firmware changes. Here is what that entails:

Invert the endstops. This involves setting X, Y, and Z minimum endstop inverting to true to invert the logic of the endstops.

Next, handle the DRV8825 Driver Types. Uncomment these and replace them with DRV8825. This applies to the X, Y, Z, Z2, and E1 driver types. Note that the E0 driver type remains as A4988.

Double the step count for the DRV8825 from 1 over 16 to 1 over 32 steps. This comes from the previous Marlin configuration which was set to 80, 80, 400, and 100 for the axis steps per unit. I've kept the extruder on an A4988 chip as the heat was building up too much on it. Calibrate to 0.84 volts. The new axis steps per unit should be set to 160, 160, 800, and 100.

Invert the X and Y directions, as well as the extruder direction.

Set the Z travel limit to 180.

Enable SD Card Support.

Enable LCD Support for the RepRap Discount Full Smart Controller.

Calibrate the DRV8825. The formula is VREF equals Rated motor current divided by 2. So a standard NEMA 17 motor is rated at 1.68 amps. Divided by 2, it is 0.84 volts. I burned out an extruder stepper following this. I dropped it to 0.6 volts. The steppers were less hot and there was no performance difference.

Replace the display. The connectors are backwards for the standard RepRap Discount Full Smart Controller. Some Amazon reviews pointed this out.

Replace the power supply. I found a 30-amp power supply on Amazon for twenty dollars. The current 20-amp power supply is underpowered. It is a known issue with these printers.

Replace the Y bracket.

Build an easy access case for the MKS Gen board.

Install a filament sensor.

Create a cover and mount for the power supply.

I tested manual LCD bed leveling using Mesh Bed Leveling, Probe Manually, and LCD Bed Leveling, but ended up reverting it.

Implement Auto Bed Leveling using a Z Probe Servo. This involves wiring and Marlin Configuration. You can find helpful videos and documentation online for this. In the firmware, uncomment the following lines: Define Auto Bed Leveling Linear, set the number of servos to 3, set the Z probe servo number to 0, define the Z servo angles as 80 and 0—this one is important on the servo arm deploy and retraction, use the Z max plug since I have an extra z max to use for the switch on the servo arm, set the Z clearance between probes to 10, set the Z clearance for multiple probes to 10, set the Z probe low point to negative 10, which is the farthest distance below the trigger point to go before stopping, set Z min probe to use the Z min endstop pin, and set the Z min probe pin to 19. Pin 32 is the RAMPS default, but we are using an MKS Gen 1.4 Board.

M280 is important. This controls the servo. Use this for testing. Use the values gathered here to set the Z servo angles property. To get servo 0 angle, enter M280 P0. To move servo 0 to a 90-degree angle, enter M280 P0 S90. To move servo 0 to a 0-degree angle, enter M280 P0 S0.

Set the Marlin config of zprobe offset to negative 1, negative 40, and negative 7.7.

Use M420 to get or set the bed level state.

Use M119 to check endstop states.

Use G29 to start the auto bed leveling routine.
