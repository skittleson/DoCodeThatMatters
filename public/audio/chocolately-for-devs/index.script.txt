As a software engineer, you know how often you need to start with a clean slate in an operating system. Usually, the solution is to use containers for development, but even your primary OS can suffer from what's called "bit rot" over time. For Windows, a factory reset gets things back into a fresh state.

But let me tell you, I've got a lot of apps that I use on a regular basis, and installing them all by hand could take up to a whole day. Not anymore, though! So here's a PowerShell script that I've put together to get the common apps that I need installed quickly.

To make this work, first you'll need to install something called Chocolatey. You can find the link on the blog post for more information on how to do that. Once it's set up, we can move on to installing these packages.

The script is a long list of apps that I use regularly: 7zip, Arduino, Audacity... you get the idea. The idea is to install all of these at once with just one command. Now, if you want to see what other local packages are available, you can run this command instead: `choco list --localonly`.

You'll find more details on how Chocolatey works and more information on the different apps I've listed by checking out the full post on our blog.
